## 项目概述

一个基于 Python Tkinter 的图形界面应用程序，用于分析材料的拉伸测试数据。系统能够自动计算屈服强度、抗拉强度和延伸率，并提供可视化应力-应变曲线。

## ✨ 主要功能

1.多Sheet数据支持 - 同时处理Excel文件中多个测试样本数据

2.智能参数配置 - 自动保存和加载测试参数（横截面积等）

3.鲁棒性计算算法 - 采用多种方法计算屈服强度，提高准确性

4.数据可视化 - 实时绘制应力-应变曲线，支持多曲线对比

5.结果导出 - 一键导出所有计算结果和图表

6.图例编辑 - 自定义图表显示文本

## 📊 数据格式要求

### Excel 文件格式

程序支持标准的Excel文件（.xlsx, .xls），数据应包含以下列：

载荷    拉伸过程中的载荷数据    N

引伸计    位移或应变数据    mm

注意：程序会自动识别包含"载荷"、"Load"、"引伸"、"Extenso"等关键词的列

### 配置文件

系统会生成同名的CSV配置文件，自动保存每个Sheet的横截面积参数：
```javascript
csv

sheet_name,cross_sectional_area

Sheet1,25.0

Sheet2,24.8
```
## 🔧 使用指南

### 加载数据
点击"加载Excel数据"按钮选择文件

程序自动识别所有Sheet和测试数据

### 设置参数
在"测试参数输入"区域为每个Sheet输入横截面积

点击"确认所有参数"保存设置

### 数据分析
处理当前Sheet：分析选中的单个样本

批量处理所有Sheet：一次性分析所有样本

### 结果查看
"计算结果"区域显示力学性能参数

图形区域显示应力-应变曲线

### 导出结果
保存图表：导出PNG/PDF格式的应力-应变曲线

导出所有结果：保存计算结果为Excel/CSV/TXT格式
